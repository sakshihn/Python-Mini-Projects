# Modular Grade Calculator 🎓

A simple Python mini project that calculates the **average marks of a student** using **functions and lambda expressions**.

## 📌 About the Project

This program allows a user to:

- Enter the number of subjects
- Input marks for each subject
- Store the marks in a list
- Calculate the average marks

The program demonstrates **modular programming using functions** and **lambda for quick calculations**.

## 🚀 Features

- Uses **functions for code reusability**
- Stores data using **lists**
- Collects input using **loops**
- Calculates average using a **lambda function**

## 🛠 Concepts Used

- Python Functions
- Lambda Functions
- Lists
- Loops
- Input / Output

## ▶️ How to Run

1. Clone the repository or download the project.
2. Navigate to the project folder.
3. Run the Python file:

```bash
python calci.py
```
