# 🔢 Number Analyzer

A simple Python program that analyzes a list of numbers entered by the user.

The program takes numbers separated by spaces and calculates useful statistics like sum, average, maximum, minimum, and separates even and odd numbers.

---

## ✨ Features

- Accepts multiple numbers as input
- Calculates total numbers
- Finds the sum of all numbers
- Calculates the average
- Finds the maximum and minimum values
- Separates even and odd numbers
- Handles invalid input using `try` and `except`
- Allows the user to run the program multiple times using a `while` loop

---

- Python
