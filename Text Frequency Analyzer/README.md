# 📊 Text Frequency Analyzer

A simple Python mini project that analyzes a piece of text and counts how many times each word appears.

The program takes a sentence or paragraph from the user, processes the text, and displays the frequency of each word along with the most frequently used word.

## 🔹 How It Works

- Takes text input from the user
- Converts the text to lowercase
- Removes punctuation from the text
- Splits the text into individual words
- Uses a set to identify unique words
- Uses a dictionary to store word frequencies
- Displays the frequency of each word
- Finds and prints the most frequent word

## 🧠 Concepts Used

- Strings and string methods (lower(), replace(), split())
- Lists
- Sets
- Dictionaries
- Loops
- Basic Python logic

## ▶️ Example

Input:
Python is easy and Python is powerful

Output:
python : 2
is : 2
easy : 1
and : 1
powerful : 1

Most frequent word: python with frequency: 2
