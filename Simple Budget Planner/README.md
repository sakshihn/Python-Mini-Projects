## 💰 Hosteller Budget Planner

## 📌 About
A simple Python program that helps hostellers calculate their **total monthly expenses** by entering costs like rent, food, groceries, shopping, and accessories.

## ⚙ How It Works
1. The user enters different monthly expenses.
2. Food and groceries are combined as total food expenses.
3. The program calculates and displays the **total monthly spending**.

## ▶ Run the Program
Save the file as `budget_planner.py` and run:

python budget_planner.py

Enter your expenses and the program will show your **total monthly expense**.

✨ A small Python program to manage hostel life expenses easily!
