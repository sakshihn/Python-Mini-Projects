# Mini Sentiment Counter

A simple Python mini project that checks whether a review is **Positive, Negative, or Neutral** based on keyword matching.

## How it Works
1. User enters a review.
2. Text is converted to lowercase.
3. The sentence is split into words.
4. Words are compared with predefined **positive** and **negative** lists.
5. The program counts matches and determines the sentiment.

## Example

Input:
This movie was amazing but the ending was bad

Output:
Positive count: 1  
Negative count: 1  
The statement is Neutral!

## Tech Used
- Python
- Lists
- Loops
- Conditional statements

## Purpose
Practice basic **Python logic, loops, and simple sentiment analysis**.
