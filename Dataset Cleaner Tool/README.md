# Dataset Cleaner Tool 🧹

A simple Python mini project to clean messy datasets for analysis.

---

## Features ✨

- Remove extra spaces  
- Convert text to lowercase  
- Remove empty values  
- Print cleaned data in a readable format  

---

## How to Run 🚀

1. Save the script as `dataset_cleaner.py`  
2. Run in terminal or IDE:
```bash
python dataset_cleaner.py
```
3. Enter your dataset when prompted
